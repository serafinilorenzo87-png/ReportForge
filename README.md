# ReportForge

A local-first desktop application for pentest reporting, built for security professionals.

## 🚀 Features
- Project-based pentest reporting
- Structured findings with severity levels
- Evidence management
- Automated report generation (DOCX/PDF)
- CVE / CWE references support

## 🛠️ Tech Stack
- Python
- PySide6 (GUI)
- SQLAlchemy (ORM)
- SQLite

## ⚡ Quick Start

```bash
git clone https://github.com/TUO_USERNAME/ReportForge.git
cd ReportForge
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py



ReportForge is a local-first desktop application for Windows that helps security learners and pentesters write structured, professional English pentest reports while keeping all data stored locally.
