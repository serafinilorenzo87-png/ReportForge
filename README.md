\# ReportForge



ReportForge is a local-first desktop application for Windows that helps security learners and pentesters write structured, professional English pentest reports while keeping all data stored locally.



\## MVP goals

\- Create a local project

\- Add findings manually

\- Organize report sections

\- Export reports in Markdown



\## Planned features

\- Methodology Checklist

\- References for each finding

\- NVD/CVE and CWE support

\- Clean desktop UI

\- Local-only storage
